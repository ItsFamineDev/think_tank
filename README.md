**Command Stuff**

```

first hit the wind chime, then run this command

/execute in minecraft:themaze run tp @a -91.13 65.00 37.80

---If you want to restart the map:---

1.
/scoreboard players reset @a

2. 

/execute in minecraft:overworld run tp @a 197 90 -165

3.

go back to the dock and fish up or drop a puffer fish on the ground

--- If you want to teleport to a specific location for fun ---

Tp back to overworld house spawn
/execute in minecraft:overworld run tp 197 90 -165

/execute in minecraft:tightrope run tp 539 -77 55

/execute in minecraft:thebetween run tp 121 71 379

/execute in minecraft:dark_forest run tp @s 25.47 173.00 53.62 90 45

Tp to the warren
/execute in minecraft:thewarren run tp 443 -492 6

Tp to kings foyer
/execute in minecraft:kings_foyer run tp @s -11 -915 -1 90 1

Tp to upsidedown

/execute in minecraft:upsidedown run tp 312 361.5 358

Tp to the tank

/execute in minecraft:thetank run tp @s 68 216 64

Tp to the maze

/execute in minecraft:themaze run tp @s 206 65 37

Tp to the fractal

/execute in minecraft:fractal run tp @s 10 -785 7 90 30
```
